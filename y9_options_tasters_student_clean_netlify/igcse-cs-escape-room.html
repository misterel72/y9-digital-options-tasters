Source Pack: Rogue AI Training Facility

System message

ACCESS DENIED. The facility has been locked by the training AI. To escape, you must collect six keys by solving problems using computational thinking.

Each room gives you evidence. Use the source material carefully. The AI is literal, petty and has the emotional range of a printer error message.

Reference card: Binary place values

Binary uses place values based on powers of 2.

1286432168421
01000001

01000001 = 64 + 1 = 65. In ASCII, 65 can represent A.

Reference card: Test data

  • Normal data: data the program should accept.
  • Boundary data: data at the edge of what is allowed.
  • Erroneous data: data the program should reject.

For an age input that accepts 11 to 16: normal could be 13, boundary could be 11 and 16, erroneous could be 10, 17 or text.

Logic gate rules

AND: TRUE only if both inputs are TRUEOR: TRUE if at least one input is TRUENOT: flips TRUE to FALSE or FALSE to TRUE
